# My first Git repo
## Опис
Навчальний репозиторій для тренування базових навичок роботи з Git та GitHub.

## Призначення
Тестування комітів, гілок та злиття через SSH-з’єднання.

## Структура
- **src/** — вихідний код  
- **docs/** — документація  
- **assets/** — додаткові матеріали  
- **README.md** — опис репозиторію
